Welcome to Our API

User Routes
POST /api/users/register
Register a new user
POST /api/users/changepassword
Change user password (requires authentication)
POST /api/users/login
User login
POST /api/users/logout
User logout (requires authentication)
GET /api/users/verify
Verify user session
GET /api/users/:id
Get user by ID (requires authentication)
GET /api/users/user/role
Get user role (requires authentication)
PUT /api/users/:id
Update user (requires authentication)
Product Routes
GET /api/products
Get all products
GET /api/products/search
Search products
GET /api/products/:id
Get product by ID
POST /api/products
Create a new product (requires seller authentication)
PUT /api/products/:id
Update a product (requires seller authentication)
DELETE /api/products/:id
Delete a product (requires seller authentication)
GET /api/products/seller/listings
Get product listings by user (requires authentication)
GET /api/products/:id/listings
Get product listings
POST /api/products/seller
Create seller products with listings (requires seller authentication)
DELETE /api/products/seller/:id
Remove product from seller listing (requires seller authentication)
Category Routes
POST /api/categories
Create a new category (requires seller authentication)
GET /api/categories
Get all categories (requires seller authentication)
GET /api/categories/:id
Get category by ID (requires seller authentication)
PUT /api/categories/:id
Update a category (requires seller authentication)
DELETE /api/categories/:id
Delete a category (requires seller authentication)
